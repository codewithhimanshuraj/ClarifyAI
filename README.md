# 🚀 ClarifyAI

ClarifyAI is a modern AI-powered web application built to simplify interactions with intelligent systems. This project focuses on providing a clean UI, smooth user experience, and scalable architecture using the latest web technologies.

---

## ✨ Features

* 🤖 AI-powered chat functionality
* ⚡ Fast and responsive UI with smooth interactions
* 🌙 Dark/Light mode support
* 📊 Scalable and clean architecture
* 🔐 Secure API handling using backend routes
* 📱 Fully responsive design

---

## 🛠️ Tech Stack

* **Next.js (App Router)** – React framework for building production apps
* **TypeScript** – For type safety and better code quality
* **Tailwind CSS** – For fast and responsive UI design
* **shadcn/ui** – Reusable and accessible UI components
* **Prisma** – Database ORM
* **Zustand** – State management
* **TanStack Query** – Data fetching and caching

---

## 📁 Project Structure

```
src/
├── app/           # App router pages & API routes
├── components/    # Reusable UI components
├── hooks/         # Custom hooks
└── lib/           # Utilities & configs
```

---

## 🚀 Getting Started

```bash
# Install dependencies
bun install

# Run development server
bun run dev
```

Visit: http://localhost:3000
